AirCursor Support

AirCursor is a macOS camera-based hand gesture preview and calibration utility. It helps you tune a green command region and preview two-hand movement, click, right-click, and drag gestures.

Getting Started

  • Open AirCursor from the menu bar.
  • Allow camera access when macOS asks.
  • Use the setup wizard to select your camera, display, command region, and gesture sensitivity.
  • The Mac App Store version does not require Accessibility permission and does not control the system pointer.

Common Issues

  • If the preview does not update, confirm that AirCursor preview is enabled from the menu bar or with Command-0.
  • If the camera is blank, confirm camera permission in macOS System Settings.
  • If tracking feels unstable, rerun the setup wizard and adjust the command region, stability, and pinch distance.

Contact

For questions, bug reports, accessibility feedback, or feature requests, open a public support request: AirCursor Support Request .

You can also browse existing support requests at GitHub Issues. Please do not include private camera images, medical details, or other sensitive personal information in public issues.
